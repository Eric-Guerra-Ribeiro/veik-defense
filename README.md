# Veik Defense

Tower defense game based in a militray invasion. This project was an assignment for the CES-22 course (OOP) at Aeronautics Institute of Technology (ITA).

- [**Setup**](#setup)
- [**Assets**](#assets)
- [**Running The Game**](#running-the-game)


## Setup

### Create python virtual environment
  ```
  virtualenv venv
  ```

### Activate virtual environment (Windows)
  ```
  cd venv/Scripts/
  ./activate
  ```

### Activate virtual environment (Ubuntu)
  ```
  source venv/bin/activate
  ```


### Dependency installation
  ```
  pip install -r requirements.txt
  ```
